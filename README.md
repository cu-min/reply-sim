# 如果这样回 · 微信小程序 MVP 第一版

这是一个基于原生微信小程序的本地 `mock` MVP 项目，当前以“可运行基线”作为第一版归档。

## 当前已完成
- 5 个核心页面已落地：
  - `首页`
  - `剧本详情页`
  - `聊天页`
  - `结局页`
  - `我的页`
- 页面主链路已跑通：
  - `首页 → 剧本详情 → 聊天页 → 结局页 → 我的`
- 本地 `mock` 数据、基础 service 分层、基础组件已建立
- 全局设计 token 已建立，视觉风格已统一到当前版本
- 聊天页已升级为分层表达流程：
  - 先选策略
  - 再选该策略下候选回复
  - 再微调发送
- 结局页已改为连续内容流式总结页
- 本地会话恢复、历史记录、基础统计与结局沉淀已具备雏形

## 当前未接入
- 微信登录
- 云函数 / 云数据库
- 云端同步
- 真实模型接口
- 真实分享能力
- 支付 / 心动值真实逻辑

## 本地运行
1. 使用微信开发者工具打开项目根目录。
2. AppID 可先使用测试号或开发者工具提供的体验配置。
3. 直接编译即可查看当前 MVP 第一版。

## 仓库约定
- 当前版本以微信开发者工具可运行的 `JS` 文件为准。
- 历史遗留 `.ts` 文件暂不纳入第一版提交，后续统一整理。
- `project.private.config.json` 视为本地私有配置，不进入版本库。
