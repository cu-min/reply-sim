# 图标与占位资源规格

当前仓库内的 tabbar 图标与分享图标仍为占位资源，后续需要按下面规格替换正式资产。

## Tabbar

- `miniprogram/assets/tabbar/home.png`
  - 81x81 PNG
  - 透明背景
  - 线性小房子，未选中态
- `miniprogram/assets/tabbar/home-active.png`
  - 81x81 PNG
  - 透明背景
  - 实心小房子，主色 `#8B5A5A`
- `miniprogram/assets/tabbar/profile.png`
  - 81x81 PNG
  - 透明背景
  - 线性人物轮廓，未选中态
- `miniprogram/assets/tabbar/profile-active.png`
  - 81x81 PNG
  - 透明背景
  - 实心人物轮廓，主色 `#8B5A5A`

## 页面图标

- `miniprogram/assets/icons/share.png`
  - 48x48 PNG
  - 透明背景
  - 线性分享图标
  - 颜色建议 `#8B8178`

- `miniprogram/assets/icons/default-avatar.png`
  - 100x100 PNG
  - 透明或浅米底
  - 中性默认头像占位
  - 风格与整体暖米白配色一致
