# 《如果这样回》项目清单

## 当前定位
- 产品形态：微信小程序 MVP
- 产品定位：情感模拟娱乐，不是实时聊天辅助工具
- 当前阶段：前端可运行基线已完成，进入后续能力补全前的归档阶段

## 已完成
- 原生微信小程序工程骨架已搭建完成
- 5 个核心页面已落地：
  - 首页
  - 剧本详情页
  - 聊天页
  - 结局页
  - 我的页
- 首页到我的主链路已跑通：
  - 首页 → 剧本详情 → 聊天页 → 结局页 → 我的
- 底部导航已完成：
  - 首页
  - 我的
- 本地 mock 数据、service 分层、基础组件已建立
- 全局设计 token 已建立并统一到当前视觉风格
- 首页、聊天页、结局页、我的页已经历一轮结构与 UI 重构
- 聊天页已升级为分层表达流程：
  - 第一步先选策略
  - 第二步选当前策略下候选回复
  - 第三步微调后发送
- 结局页已从信息卡面板重构为连续内容流
- 本地会话恢复、历史记录、基础统计、结局沉淀已具备雏形
